# v1.1.2 リリースノート

## 主な機能

- **機密情報のサニタイズ機能** ([#68](https://github.com/tominaga-h/jarvis-shell/issues/68)): API キーやトークン値などの機密情報が履歴や AI コンテキストに流出するのを防ぐサニタイズ機能を実装
- **AI temperature 設定** ([#66](https://github.com/tominaga-h/jarvis-shell/issues/66)): `config.toml` の `[ai]` セクションで `temperature` パラメータを設定可能に

## バグ修正

- **エイリアス解除後のコマンド履歴バグ** ([#65](https://github.com/tominaga-h/jarvis-shell/issues/65)): エイリアスが解除されたコマンドが履歴に残り続けるバグを修正

## 内部改善

- version バッヂにリリースページへのリンクを追加 ([#64](https://github.com/tominaga-h/jarvis-shell/issues/64))
- 新しいデモ GIF を作成 ([#63](https://github.com/tominaga-h/jarvis-shell/issues/63))
- release コマンドの手順を調整

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
