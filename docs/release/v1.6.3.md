# v1.6.3 リリースノート

## 主な機能

- なし（バグ修正リリース）

## バグ修正

- **タイポ補正で n を押すと AI が走る問題の修正**: タイポ補正プロンプトで n (Reject) を選択すると、元の入力が NaturalLanguage として AI に送信されていた問題を修正。n を押した場合はプロンプトに戻るよう変更
- **Ctrl+C でプロセスが終了する問題の修正**: タイポ補正や自動調査の確認プロンプトで Ctrl+C を押すとシェルプロセスごと終了してしまう問題を修正。SIGINT を一時的に無視し、Ctrl+C をキャンセル操作として扱うよう変更

## 内部改善

- `read_line_ignoring_sigint()` ヘルパー関数を導入し、対話プロンプトでの SIGINT ハンドリングを共通化

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
