# v1.4.0 リリースノート

## 主な機能

- **AI の賢さ改善** ([#85](https://github.com/tominaga-h/jarvis-shell/issues/85)): プロンプトの改善、再調査時のコンテキスト引き継ぎ、`extract_shell_command` のショートサーキット修正、`search_replace` ツールの追加を実施
- **ブランチ補完の設定可能化** ([#84](https://github.com/tominaga-h/jarvis-shell/issues/84), [#86](https://github.com/tominaga-h/jarvis-shell/issues/86)): `config.toml` の `completion.git_branch_commands` でブランチ補完対象の Git サブコマンドをカスタマイズ可能に

## バグ修正

- `extract_shell_command` のショートサーキット処理を修正 ([#85](https://github.com/tominaga-h/jarvis-shell/issues/85))

## 内部改善

- AI のコンテキスト認識の改善
- Cargo.lock の更新

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
