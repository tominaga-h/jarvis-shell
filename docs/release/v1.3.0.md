# v1.3.0 リリースノート

## 主な機能

- **セッション機能** ([#75](https://github.com/tominaga-h/jarvis-shell/issues/75), [#78](https://github.com/tominaga-h/jarvis-shell/issues/78)): セッション ID によりコマンド履歴やログを分割するセッション管理機能を実装
- **`-c` オプション** ([#81](https://github.com/tominaga-h/jarvis-shell/issues/81)): `jarvish -c "command"` のように、コマンド文字列を引数で渡して実行するオプションを実装
- **自動調査のコマンド除外設定** ([#82](https://github.com/tominaga-h/jarvis-shell/issues/82)): `config.toml` の `ignore_auto_investigation_cmds` で特定のコマンドの自動調査を無効化可能に

## バグ修正

- ブランチ補完で現在のブランチが先頭に表示されない問題を修正 ([#76](https://github.com/tominaga-h/jarvis-shell/issues/76))
- テストが失敗する問題を修正

## 内部改善

- テスト設定の調整（entire を enable）

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
