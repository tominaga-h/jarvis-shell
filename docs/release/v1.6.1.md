# v1.6.1 リリースノート

## 主な機能

- なし（バグ修正リリース）

## バグ修正

- **非対話モード自動調査暴走の修正**: `-c` オプションによる非対話モード実行時に自動調査が暴走し、AI 失敗時にカスケード障害が発生する問題を修正
- **Starship プロンプトのレンダリングバグ修正**: Starship 使用時にプロンプトのレンダリングが崩れる問題を解消

## 内部改善

- Starship プロンプトのキャッシュ実装によりプロンプト描画のパフォーマンスを向上
- リリースコマンド実行時に Homebrew Formula の更新ステップを追加

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
