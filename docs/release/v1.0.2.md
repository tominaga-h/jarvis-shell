# v1.0.2 リリースノート

## 主な機能

- **Markdown レンダリング対応** ([#27](https://github.com/tominaga-h/jarvis-shell/issues/27)): AI の応答を Markdown としてターミナル上でレンダリング表示
- **Markdown レンダリングの設定オプション**: `config.toml` の `markdown_rendering` で有効/無効を切り替え可能に
- **`--version` / `-v` オプションの追加** ([#52](https://github.com/tominaga-h/jarvis-shell/issues/52)): `jarvish --version` でバージョン情報を表示
- **Git エイリアスでのブランチ補完** ([#54](https://github.com/tominaga-h/jarvis-shell/issues/54)): Git エイリアス（例: `g checkout`）でもブランチ名の補完が動作するように

## バグ修正

- **CPU 使用率のバグを修正** ([#47](https://github.com/tominaga-h/jarvis-shell/issues/47)): アイドル時の CPU 使用率が高くなる問題に対処
- **PATH キャッシュ問題をキャッシュレス化で解決** ([#51](https://github.com/tominaga-h/jarvis-shell/issues/51)): Fish Shell の設計思想に倣い、PATH の解決を都度 `which` で行うように変更
- **`source` コマンドの出力改善** ([#55](https://github.com/tominaga-h/jarvis-shell/issues/55)): `source` 実行時の出力結果を改善し、表示を変更

## 内部改善

- リリースコマンド（`/release`）の作成
- README の整備（ロゴ削除、不要な絵文字の削除、デモ GIF の追加）

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
