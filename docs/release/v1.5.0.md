# v1.5.0 リリースノート

## 主な機能

- **Starship プロンプト連携** ([#57](https://github.com/tominaga-h/jarvis-shell/issues/57)): [Starship](https://starship.rs/) をプロンプトのカスタマイズ手段としてネイティブサポート。`config.toml` で `prompt.starship = true` を設定するだけで、既存の Starship 設定をそのまま利用可能。`--status`、`--cmd-duration`、`--terminal-width` を `starship prompt` に渡すため、主要な Starship モジュールが正しく動作する。前提条件を満たしていない場合はビルトインプロンプトに自動フォールバック

## バグ修正

- なし

## 内部改善

- プロンプトモジュールのリファクタリング: `mod.rs` から `JarvisPrompt` を専用ファイル (`jarvis.rs`) に分割し、モジュール構造を整理

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
