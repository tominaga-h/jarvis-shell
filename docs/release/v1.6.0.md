# v1.6.0 リリースノート

## 主な機能

- **タイポ補正** ([#83](https://github.com/tominaga-h/jarvis-shell/issues/83)): 存在しないコマンドを入力した際に、PATH 上の類似コマンドを自動検出して補正候補を提示する zsh 互換機能を追加。`🤵 jarvish: correct 'gti' to 'git' [nyae]?` のようなプロンプトが表示され、`y`/Enter で補正実行・`n` で AI ルーティング・`a`/`e` で中止を選択できる。Damerau-Levenshtein 距離（転置を距離 1 として扱う）を使用するため、隣接文字の入れ替え（`gti` → `git`）も精度よく検出可能

## バグ修正

- なし

## 内部改善

- `src/engine/typo.rs` を新規追加: Damerau-Levenshtein 距離計算・PATH 走査・コマンド名判定ロジックを独立モジュールとして実装

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
