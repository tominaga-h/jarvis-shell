# v1.1.0 リリースノート

## 主な機能

- **AI パイプ機能** ([#58](https://github.com/tominaga-h/jarvis-shell/issues/58)): コマンドの出力を `| ai "..."` で AI に渡し、自然言語でフィルタリング・整形・抽出が可能に（例: `cat access.log | ai "500番台のエラーだけ抽出して"`）
- **システムプロンプトへの README 注入** ([#59](https://github.com/tominaga-h/jarvis-shell/issues/59)): AI がプロジェクトの README を参照し、よりコンテキストを踏まえた回答を生成
- **オフライン検知バナー**: ネットワーク接続状態に応じてバナーメッセージが「All systems are operational」から動的に変化
- **AI Markdown 判定**: AI の応答が Markdown 形式かどうかを判定し、適切にレンダリングを切り替え
- **`git push` 時のブランチ補完**: `git push` コマンドでもブランチ名の補完が動作するように

## バグ修正

- **CI Workflow の重複実行を修正**: push 時と PR 作成時に Workflow が同時実行される問題を修正

## 内部改善

- AI 応答の Markdown 判定ロジックの追加
- ブランチ補完の対象サブコマンドを拡充

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
