# v1.13.1 リリースノート

> 起動時バナーのデザインを刷新したパッチリリースです。

## 内部改善

### 起動バナーのデザインを刷新しました — v1.13.1

`jarvish` 起動時に表示されるウェルカムバナーを変更しました (`src/cli/banner.rs`)。

- ASCII ロゴをよりコンパクトな新デザインに差し替え
- 配色を赤系へ変更（ロゴを赤で表示）
- 二重線セパレータと独立したバージョン行を廃止し、ロゴ幅に合わせた一本の細線セパレータの右端にバージョンタグ (`v1.13.1`) を配置するレイアウトに変更

機能・挙動への影響はありません（表示のみの変更）。

## 対応プラットフォーム

- macOS (aarch64 / x86_64)
- Linux (x86_64)

## 既知の制限

- Windows は未対応
- AI 機能の利用には OpenAI API キーが必要
