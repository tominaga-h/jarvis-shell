# v1.2.1 リリースノート

## 主な機能

- **`which` / `type` ビルトインコマンド** ([#74](https://github.com/tominaga-h/jarvis-shell/issues/74)): コマンドの種類（ビルトイン / 外部コマンド / エイリアス）やパスを確認できる `which` / `type` コマンドを実装
- **`pushd` / `popd` / `dirs` ビルトインコマンド** ([#73](https://github.com/tominaga-h/jarvis-shell/issues/73)): ディレクトリスタックを操作する `pushd` / `popd` / `dirs` コマンドを実装
- **`pwd` コマンド**: `cwd` のエイリアスとして `pwd` コマンドを追加
