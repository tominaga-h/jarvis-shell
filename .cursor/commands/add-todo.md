# /add-todo コマンド

ユーザーが入力した内容を `docs/TODO.md` の末尾に新しいTODO項目として追加する。

## 手順

1. `docs/TODO.md` を読み込む
2. ファイル内の `[#N]` パターンから最大の番号を特定する
3. 新しい番号 = 最大番号 + 1
4. ユーザーが入力した内容を `- [ ] [#新番号] 内容` の形式でファイル末尾に追加する
5. ファイル末尾には空行を1つ残すこと

## フォーマット規則

- 形式: `- [ ] [#N] 内容`
- 番号は既存の最大番号 + 1 の連番
- 末尾の空行を維持する

## 例

ユーザー入力: `/add-todo ヘルプコマンドを実装する`

追加される行: `- [ ] [#12] ヘルプコマンドを実装する`
