# /fix-todo コマンド

ユーザーが指定した番号のTODO項目を完了状態にする。

## 手順

1. `docs/TODO.md` を読み込む
2. ユーザーが指定した番号 N に該当する行 `- [ ] [#N]` を探す
3. その行の `- [ ]` を `- [x]` に変更する
4. 既に `- [x]` の場合はユーザーに「既に完了済みです」と伝える

## フォーマット規則

- 変更前: `- [ ] [#N] 内容`
- 変更後: `- [x] [#N] 内容`
- 内容やその他の行は一切変更しない

## 例

ユーザー入力: `/fix-todo 5`

変更: `- [ ] [#5] 自然言語を入力したはずなのに...` → `- [x] [#5] 自然言語を入力したはずなのに...`
